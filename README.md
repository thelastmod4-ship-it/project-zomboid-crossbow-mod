# Medieval Crossbow Mod para Project Zomboid B42.19

Um mod que adiciona uma besta medieval funcional com sistema de flechas ao Project Zomboid.

## 📚 Características

- **Besta Medieval**: Arma poderosa de ataque à distância (Dano: 6.0-8.5)
- **Flechas Medievais**: Munição reutilizável com limite de durabilidade
- **Sistema de Crafting**: Fabrique bestas e flechas com materiais comuns
- **Recarga Realista**: Velocidade de ataque apropriada para um arco/besta
- **Distribuição em Loot**: Encontre em lojas de armas, esportes e casas

## 📶 Especificações

### Besta Medieval
- **Dano**: 6.0 - 8.5
- **Durabilidade**: 120 usos
- **Peso**: 3.5 kg
- **Tipo**: Arma de 2 mãos
- **Tipo de Munição**: Flechas Medievais
- **Recarga**: Lenta (realista)

### Flechas Medievais
- **Peso**: 0.05 kg (cada)
- **Durabilidade**: 50 usos
- **Máx. Pilha**: 60 unidades
- **Reutilizável**: Sim

## 💾 Crafting

### Fabricar Flechas Medievais
- **Materiais**: Cabo de Flecha (1x) + String (1x)
- **Ferramentas**: Martelo
- **Habilidade**: Crafting 2+
- **Tempo**: 5 minutos

### Fabricar Besta Medieval
- **Materiais**: Plank (3x) + String (2x) + Scrap Metal (5x)
- **Ferramentas**: Martelo + Serra
- **Habilidade**: Crafting 4+
- **Tempo**: 20 minutos

## 🌳 Onde Encontrar

- **Gun Stores**: Bestas e Flechas
- **Sporting Goods**: Bestas e Flechas
- **Casas Residenciais**: Flechas
- **Garagens**: Bestas

## 📦 Instalação

1. Baixe ou clone este repositório
2. Copie a pasta `project-zomboid-crossbow-mod` para:
   ```
   C:\Users\[SEU_USER]\Zomboid\mods\
   ```
   (ou `~/Zomboid/mods/` no Linux/Mac)

3. Renomeie a pasta para `MedievalCrossbow` (opcional mas recomendado)

4. Inicie o Project Zomboid

5. Na tela de seleção de mundo, clique em "Mods"

6. Procure por "Medieval Crossbow Mod" e ative-o

7. Crie seu mundo e aproveite!

# Medieval Crossbow Mod para Project Zomboid

Um mod que adiciona uma besta medieval funcional com sistema de flechas ao Project Zomboid.

## Características

- **Besta Medieval**: Arma poderosa de ataque à distância
- **Flechas**: Munição reutilizável com limite de durabilidade
- **Sistema de Recarga**: Mecânica realista de recarga
- **Dano Escalável**: Dano baseado em habilidade de armas de fogo
- **Crafting**: Fabricação de flechas e manutenção

## Instalação

1. Baixe o mod
2. Coloque a pasta `MedievalCrossbow` em `Zomboid/mods/`
3. Ative o mod na seleção de mods ao iniciar o jogo

## Uso

- A besta pode ser encontrada em áreas específicas ou fabricada
- As flechas são munição reutilizável
- Use como qualquer outra arma de fogo no jogo
